# Структура базы знаний SAP

Данная структура предназначена для организации базы знаний по SAP, разделённой на инциденты, изменения и шаблоны.

## Каталоги

- `incidents/` — инциденты по модулям SAP
- `changes/` — изменения, включая стандартные и крупные проекты
- `templates/` — шаблоны документов для инцидентов и изменений

## Подкаталоги

### incidents/
- SD/ — Сбыт (Sales & Distribution)
- MM/ — Управление материалами
- FI/ — Финансы
- CO/ — Контроллинг
- PP/ — Планирование производства
- WM/ — Управление складом
- QM/ — Управление качеством
- PM/ — Техобслуживание
- HR/ — Персонал
- BASIS/ — Базис
- INTEGRATION/ — Интеграции

### changes/
- standard-changes/ — стандартные изменения по модулям
- changes/ — крупные изменения и проекты

### templates/
- incident-template.md — шаблон инцидента
- standard-change-template.md — шаблон стандартного изменения
- change-template.md — шаблон изменения
